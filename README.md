# Tic Tac Toe (C++)

## Description
This is a console-based Tic Tac Toe game developed in C++ as a university project. Two players can play the game by entering their names and choosing positions on a 3×3 game board. The program checks for a winner after every move and declares the result.

## Features
- Two-player gameplay
- Player name input
- X and O tokens
- Win detection for rows, columns, and diagonals
- Draw (tie) detection
- Prevents players from selecting an occupied position
- Console-based interface

## Technologies Used
- C++
- Standard Library (`iostream`, `cstdlib`)

## How to Run
1. Open the project in a C++ IDE such as Dev-C++, Code::Blocks, or Visual Studio.
2. Compile the source code.
3. Run the program.
4. Enter both player names.
5. Follow the on-screen instructions to play.

## Project Status
This project was created for learning C++ programming concepts and demonstrates the use of arrays, functions, loops, conditional statements, and switch-case.
